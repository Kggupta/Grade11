Overview

Swift is a new programming language created by Apple which is in the C family of coding languages. It is a language that is very lightweight and fast with a simpler and stronger structure than many other mainstream languages. This language is used primarily for macOS, iOS, watchOS and tvOS. In my personal experience it is interactive and fun with a simple syntax and an excelent error detection compiler.

Before Swift 4 there was of course Swift 3, so before we start it is important to know what is new in the most recently released version. Below is a list of the main changes from Swift 3 to 4.

  • Refreshed String Manifesto

    In version 4, the string type conforms to the ‘Collection’ protocol. This way the programmer can just enumerate over a string to get all characters.


  • Enhanced File Paths

    Swift 3 had the new addition of File Paths, but their usage was limited, and did not work well when using structs. In Swift 4 a program can find the file path of a referenced object without the input of the user, thus saving time becuase you dont need to copy in the FULL file path.


  • Generic Subscripts

    In other versions of Swift, the developer had to define the type of value to be returned in a funtion. Instead, with Swift the system will infer what the function can return in any iteration and decide the variable type on its own.


  • Better Variable Declaration System

    In most programs variables must always be declared BEFORE they are used. In Swift 4 a new system was introduced where a variable can be declared anywhere in the code and the compiler will still recognise the use of it without needing it to be at the very top of the program. It has also introduces a feature where the type of the variable need not be stated in the declaration as it is decided by the software as the code is run
