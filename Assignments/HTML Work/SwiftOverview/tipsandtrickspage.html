Tips and Tricks:

How to program like a professional!

Clicking any swift logo will return you to the homepage.
  1. Keep track of your build time when compiling your Swift code. A sudden spike in build time could mean your code needs to be revized and made smaller! Try to keep your build time less than 5 seconds


  2. Use the IOS simulator in full screen mode! The simulator takes up enormous amounts of space as it acts as an entire phone, so making it full screen helps to allow the computer to allocate more memory to the application as the other running apps are put into the background.   4. Easily move a large section of code up a line by using CMND+OPTION+{
    To move a large section of code down a line use CMND+OPTION+}
  5. Toggle the debug console using CMND+SHIFT+Y
  6. Show/Hide the assistant editor using CMND+OPTION+RETURN
  7. In Xcode, a file can be opened in many ways: in the standard editor, in a new or existing assistant editor, in a new or existing tab, or in a new window. Click SHIFT+OPTION+CMND to open the popup and select the area of the screen you want the program to run.

  8. If you ever find yourself typing the same peices of code again and again use CMND+OPTION+CTRL+2. This will activate the autofil for whatever command you are using and save you mounds of time.

  9. Finally, the best trick there is with Swift 4 is Automated Testing. With the apps that I have put on the app store many of them were more than 4 thousand lines of code. It would be physically imposible to test the code for every condition every time I added something new to the code! With automated testing the simulator will run all the tests for you can return a PASS or FAIL. Using this trick saved me hours upon hours of time so that i could further my projects faster.
