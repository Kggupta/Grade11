Variables

Use var for variables that can change which is known as mutable and let for constants that can't change, this is non-mutable.
var varIntName: Int = 7 Integers are "whole" numbers, numbers without a fractional component. Setting a Integer as a fraction would crash your program. This is usefull for saving space when you know a variable will never hold a number that is not "whole"
var varFloatName: Float = 1.987 Floats are decimal-point numbers, therefore are numbers with a fractional component. They're single precision, with as little as 6 decimal digits.
let constDoubleName: Double = 21.6598301459 Doubles are floating point numbers with double the precision, they have a fraction component to them. In addition, doubles are preferred over floats when used in application.
var varStringName: String = "Hello World" A String is a sequence of characters, such as text. Keep in mind that strings can also carry numbers just like any of the variable types above, BUT they cannot do opperations without special steps beforehand.
var varBoolName: Bool = false Booleans are used for logical operations. A boolean can be either true or false.
var varName = "This is an example of using Swift's type inference capabilities" Swift can infer the type of variable that is being applied to it (Int, Double, Float, String, Etc). In this case varName is being infered as a String


Functions

Functions are used as 'containers' in Swift code. It has an input (parameters/objects) and outputs (returns). Functions are usually to create abstractions in code, so that the snippet inside the funtion becomes reusable. Below is an example of how a function is setup. 

            func funcName (var1: String, var2:String) -> String
                {
                    return "\(var1), \(var2)"
                }
The reason why functions are so great is that once you code them, you can call them as many times as you want after! (amazing way to save time)

Control Flow
Below is an if-statement, or conditional. You use them to make decisions based on logic. 

            if var1  ==  var2
                {
                    print("The condition was true")
                } else {
                    print("The condition was not true")
                }
There are numerous types of 'operands' such as the >, <, !, and & symbols

Loops
Loops are used to repeat the same code in the defined parenthesis for any amount of iterations 

            for i in 1...5 
                {
                    print(i)
                }
The code above would print 1 to 5 (including 5). You can also use the less that symbol before the second variable like this: a.. <b to loop from a to b while not including b. 

            while b <= 60 && b > 0
                {
                    print(b)
                    b -= 1
                }
In addition a while loop can be used when the exact number of iterations is unknown. Above is an example of this.

Arrays
Arrays are a collection type. It is a type of special variable that can hold multiple values known as cells and cells always contain elements of the same type. 

            var arrayName = ["Cell0", "Cell1", "Cell2", "Cell3"]
Items can be added to the array: 

            arrayName += ["Cell4"]
And finally you can get the contents of a cell using subscript syntax. 

            let Cell5 = arrayName[4]
Keep in mind that arrays are 'zero-index', so the number of the first cell is 0 NOT 1.
